# PRD: SalesFit

## 목표
가전제품 상담원이 상담 중 실시간 AI 코칭을 받고, 상담 후 복기 리포트로 스스로 역량을 개선할 수 있게 한다.

## 사용자
가전제품 매장의 영업 상담원 (iOS / Android 스마트폰 사용)

## 핵심 기능
1. **실시간 녹음 + 자막**: 상담 시작 시 앱이 대화를 녹음하고 텍스트로 변환해 화면에 표시
2. **실시간 AI 코칭**: 대화 내용을 분석해 상담원에게 텍스트 팁을 실시간으로 제공
3. **복기 리포트**: 상담 종료 후 4개 항목의 상세 피드백 리포트 자동 생성
   - 고객 니즈 파악률 + 추천 멘트
   - 제품 설명 명확도
   - 클로징 타이밍
   - 아쉬운 점 + 개선 제안

## MVP 제외 사항
- 팀/관리자 계정 및 상담원 관리 기능
- 클라우드 동기화 (AsyncStorage 로컬 저장으로 대체)
- 음성 코칭 (텍스트 코칭만 제공)
- 제품 DB 연동
- 로그인/회원가입
